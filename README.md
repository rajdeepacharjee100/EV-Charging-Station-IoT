## Description
This project is an IoT-based EV Charging Station using ESP32.
It supports user authentication, payment confirmation, charging control,
and live status monitoring.

## Features
- ESP32 based controller
- Relay-controlled charging
- IR sensors for vehicle detection
- Web dashboard (PHP + MySQL)
- UPI QR payment system
- WhatsApp notification
- OLED display
- Voice alerts (DFPlayer / MP3 module)

## Technologies Used
- ESP32 (NodeMCU)
- PHP, MySQL (XAMPP)
- HTML, CSS, JavaScript
- Relay module
- IR sensors
- Wireless charging coils

## How It Works
1. User scans UPI QR
2. Payment is verified
3. Charging starts automatically
4. Status shown on OLED and website
5. Notification sent to user
